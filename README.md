## Примеры использования:

### 1. Модуль widget
1.1 Функция маскировки номера банковской карты или счета. Маску номера карты, где видны первые 6 цифр и последние 4 цифры остальные символы отображаются *.
```
 mask_account_card(card_or_account_number)
```
1.2 Функция принимает на вход строку с датой в формате "2024-03-11T02:26:18.671407"
    и возвращает строку с датой в формате "ДД.ММ.ГГГГ".
```
get_date(date)
```
